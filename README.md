# Edgee Tracking Tool

## Installation

```sh
yarn add edgee-tracking
```

```sh
npm install edgee-tracking
```

```html
<script
  type="text/javascript"
  src="https://unpkg.com/edgee-tracking@1.0.3/lib/index.min.js"
></script>
<script type="text/javascript">
  edgee_tracking.init();
</script>
```

## Usage

```js
import EdgeeTracking from "edgee-tracking";

EdgeeTracking.init();
```
